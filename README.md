# YarNet BPM

## Installation
1. Clone repo
2. Init virtual enviropment:

```bash
python -m venv venv
```

3. Activate ENV:

```bash
source ./venv/bin/activate
```

4. Install all dependencies:

```bash
pip install -r requirements.txt
```

5. Start `dev` server:

```bash
python manage.py runserver 0:8000
```
