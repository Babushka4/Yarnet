https://djbook.ru/rel1.7/howto/initial-data.html - руководство по автозаполнению БД django.

Fixtures - папка, где django по умолчанию ищет данные для автозаполнения БД.
initial_data - файлБ из которого данные берутся по умолчанию при каждой минрации (для отмены автозаполнения при миграции переименовать на _initial_data).

Для загрузки данных в БД НЕ из initial_data выполните команду manage.py loaddata <fixturename>, где <fixturename> это имя созданного вами файла с данными. Каждый раз 
при запуске loaddata, данные будут считываться из файлов и записываться в базу данных.
